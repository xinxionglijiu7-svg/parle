## Data Flow: Voice Conversation

```
┌──────┐     ①音声      ┌──────────┐    ②テキスト     ┌──────────┐
│ User │ ──────────────→│Web Speech│ ──────────────→ │ Hono API │
│(mic) │                │API (STT) │   POST /message │          │
└──────┘                │ fr-FR    │                 └────┬─────┘
   ▲                    └──────────┘                      │
   │                                                      │ ③ユーザーメッセージ
   │                                                      │   + 会話履歴
   │                                                      ▼
   │                                                ┌───────────┐
   │                                                │Conversation│
   │                                                │  Agent     │
   │                                                │ (Claude)   │
   │                                                └─────┬─────┘
   │                                                      │
   │                                                      │ ④フランス語応答テキスト
   │                                                      ▼
   │    ⑥音声再生     ┌──────────┐    ⑤テキスト     ┌──────────┐
   │ ◄───────────────│AudioPlayer│◄──────────────│Google TTS│
   │   (MP3)         │          │   POST /tts    │ Wavenet  │
   │                 └──────────┘   → MP3 audio  └──────────┘
   │
   │              ⑦ 両方のメッセージが MongoDB に保存
   │
   │  ── 会話終了時 ──────────────────────────────────────────
   │
   │                    ⑧全トランスクリプト
   │                         │
   │                         ▼
   │                   ┌───────────┐
   │                   │Correction │
   │                   │  Agent    │──→ ⑨ Feedback (JSON)
   │                   │ (Claude)  │      → MongoDB に保存
   │                   └───────────┘
   │                         │
   │                         └──→ ⑩ 語彙自動抽出
   │                               → Vocabulary に保存
   │
   └─── ⑪ FeedbackPanel で結果表示
```

## Database Schema Relations

```
User (1)
 │
 ├──→ (N) Conversation
 │         │
 │         └──→ (1) Feedback
 │                   ├── grammarErrors[] (embedded)
 │                   └── vocabularySuggestions[] (embedded)
 │
 ├──→ (N) Vocabulary
 │         └── SRS: reviewCount + lastReviewedAt
 │
 └──→ (N) LearningProgress
           └── 日次: streak, conversationsCompleted, grammarErrorCount
```
